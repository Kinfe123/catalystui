# Get Started

{<p className='text-base text-left font-light mt-4 mb-0 uppercase text-gray-600'>Installation</p>}

```bash
npm install @i4o-oss/catalystui
```

```bash
yarn add @i4o-oss/catalystui
```

```bash
pnpm add @i4o-oss/catalystui
```

```tsx
	return (
		<div className='flex flex-col items-center justify-center min-h-screen py-2'>
			<Button>Click Me</Button>
		</div>
	)
}
```
