### Props

| Name        | Type                  | Default | Description                   |
| :---------- | :-------------------- | :------ | :---------------------------- |
| alt      | `string`           |         | Alt text of the image            |
| ratio      | `number`           | `1`       | The desired aspect ratio            |
| src | `string` |         | URL of the image |
